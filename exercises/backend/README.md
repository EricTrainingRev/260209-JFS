### How to Practice Locally

To simulate the HackerRank environment, I recommend following this workflow for each assessment:

1. **Set a Timer:** Give yourself 60 to 90 minutes per problem.
2. **No Copy-Paste:** Try to write the Boilerplate (Repository, Controller, Service) from scratch to build muscle memory.
3. **Test via Postman:** Ensure your endpoints work as expected
4. **Do not use AI:** Try to do as much of this with your own knowledge to highlight what you still need to learn.
5. **Focus on the status codes:** Ensure that you get the status codes you expect on each endpoint.

You can reference documentation and your own examples for the first assessment if you are stuck, but you should limit it as much as you can for the next assignments.
