Before starting these extensions, you must enable **CORS (Cross-Origin Resource Sharing)** in your Spring Boot controllers so your frontend app can talk to your API.

## How to practice locally

1. **Run the Backend:** Start your Spring Boot application and verify the H2 console shows data.
2. **Set a Timer:** Give yourself 60 to 90 minutes per problem.
3. **No Copy-Paste:** Try to write the Boilerplate from scratch to build muscle memory.
4. **Do not use AI:** Try to do as much of this with your own knowledge to highlight what you still need to learn.

You can reference documentation and your own examples for the first assessment if you are stuck, but you should limit it as much as you can for the next assignments.
